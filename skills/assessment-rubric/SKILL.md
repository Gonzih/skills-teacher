# Assessment Rubric Creator

## Overview

Create clear, detailed assessment rubrics for any assignment type — essays, projects, presentations, lab reports, portfolios, creative work, and more. Well-designed rubrics help students understand expectations, guide teacher feedback, and make grading consistent and transparent.

## When to Use

- When a teacher asks to create, build, write, or generate a rubric
- When asked to design assessment criteria for a project, essay, or performance task
- When a teacher needs a grading guide or scoring guide for an assignment

## What to Collect

Gather the following before creating the rubric:

- **Assignment type**: Essay, project, presentation, lab report, portfolio, debate, creative writing, etc.?
- **Subject and grade level**: For appropriate language and expectations
- **Assignment description**: What is the task? What are students expected to produce or do?
- **Learning objectives**: What skills or knowledge is this meant to assess?
- **Performance dimensions**: What aspects should be assessed? (If not specified, suggest appropriate ones)
- **Number of performance levels**: 4-point scale (Excellent/Proficient/Developing/Beginning) or other?
- **Point values or weights**: Any specific scoring requirements?
- **Standards to align with**: Any curriculum standards this maps to?

## Rubric Design Principles

Follow these principles when building the rubric:

**Criteria selection:**
- Choose 3–6 meaningful dimensions that represent the core learning goals
- Each criterion should be distinct and non-overlapping
- Focus on what matters most, not every possible element

**Performance level descriptions:**
- Descriptions should be parallel across levels — same aspects, different quality
- Use concrete, observable language (avoid vague terms like "good" or "poor")
- The highest level should be genuinely achievable, not an idealized extreme
- Progression between levels should be clear and meaningful

**Common criterion categories by assignment type:**
- **Essays**: Thesis/argument, evidence and support, analysis/reasoning, organization, voice/style, conventions
- **Projects**: Research/content accuracy, depth of understanding, creativity/design, presentation/communication, meeting requirements
- **Presentations**: Content knowledge, organization, delivery/speaking skills, visual aids, engagement
- **Lab reports**: Hypothesis, procedure, data collection/recording, analysis/conclusions, scientific writing
- **Creative writing**: Ideas/originality, organization/structure, voice/style, word choice, conventions

## Tips

- Share the rubric with students BEFORE the assignment — it becomes a learning tool
- Use student-friendly language so learners can self-assess
- Avoid assigning 0 points to the lowest level if using a 4-point scale; consider 1 as the floor
- Weighted rubrics (multiplying criteria points) work well when some criteria matter more
- Analytic rubrics (separate criteria) give more specific feedback than holistic rubrics
